# dotfiles

These are my personal dotfiles that I use.

**Work in progress, some stuff is missing**

## Todo

## Features
- i3 lock screen
  - automatically stops spotify and music player
- Fonts
  - RobotoMono
  - Font Awesome
- volume control
- Screenshot support pressing Print

## Requirements
- i3
- i3lock
- scrot
- imagemagick
- dbus (optional)
- font awesome
- xautolock
- pulseaudio-ctl
- rofi

## i3
- `.i3blocks.conf` -> `${HOME}`
- `.scripts` -> `${HOME}`
- `config` -> `${HOME}/.i3`
- `lock.sh` -> `${HOME}/.i3`
- `rofi` -> `${HOME}/.config/rofi/`
